BrightstarDB Documentation

Starting with release 1.3 the core BrightstarDB user and developer docs are hosted at ReadTheDocs.org. We will also maintain an online version of the API documentation (generated by Sandcastle) on this site. If you want a local .CHM version of the API documentation please download the latest distribution. If you want a local copy of the user and developer docs, you can download a PDF from ReadTheDocs.org. If you are happy with online documentation, use the links in the table below to find the documentation you need.

BrightstarDB Release User/Developer Docs API Docs
Latest1 Latest at readthedocs.org N/A2
1.3 Version 1.3 at readthedocs.org Version 1.3 API Docs

1The latest version tracks the develop branch in our GitHub repository. Use this documentation if you are using BrightstarDB direct from the GitHub repository.
2Our current build hosting does not allow for publication of the HTML version of the API documentation. However it is easy to build it yourself from the sources.
