BrightstarDB Download

Full Installer Packages

A full Windows Installer packages can be found on our CodePlex project page. This package includes the binaries for the .NET 4.0 and Windows Phone SDKs as well as the BrightstarDB service and Polaris data management tool.

SDK via NuGet

You can also include BrightstarDB as an embedded database in your project via NuGet - just search for BrightstarDB. There are two versions of the BrightstarDB NuGet package. BrightstarDB provides the .NET 4.0 and Windows Phone libraries as well as the text template file required to generate a BrightstarDB Entity Framework context class. BrightstarDBLibs provides the libraries only, without the text templates.

Source code

The full source code for BrightstarDB can be downloaded or forked on GitHub.
