BrightstarDB Ecosystem

There are a number of products and applications that support or use BrightstarDB. This page describes all this cool stuff. If you have some product, application or service you would like to see added here, please contact us.

Community

We have created a forum for BrightstarDB User Group on Google Groups. This is the place to post questions, get help and provide us with feedback on Brightstar. You can also follow our blog or follow us on Twitter to keep up to date with the latest releases; events and news from the BrightstarDB team.

Applications

DataBrowser: The Data Browser is a Windows 8 application that provides a great way to navigate and explore data (ODATA, RDF, ATOM, RDFa) published on the web. The application is currently under active development. The source is maintained in github at https://github.com/BrightstarDB/DataBrowser, and we plan to make regular source code dumps and binary builds available through the Codeplex pages.

Web Content Management

Webnodes Semantic Integration Server: Webnodes CMS is an enterprise quality ASP.NET CMS with a unique semantic content technology. The Semantic Server module uses BrightstarDB to provide a SPARQL query endpoint over the data managed in the CMS. The module also offers full SDShare data integration functionality.
